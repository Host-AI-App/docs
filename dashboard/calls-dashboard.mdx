# Calls Dashboard

Spot patterns, fix gaps, and boost automation with Insights

## Overview

The calls dashboard enables users to view data about calls made by both AI systems and team members. Users can compare metrics across different time periods and filter results by individual team members.

## Calls By Member

This section allows comparison of call volumes made by each team member or AI Agent. Users can click team member names to access call transcripts. The interface displays:

- Total call counts
- Breakdown by communication channel
- Separate tracking for outgoing and incoming calls

## Manager Dashboard

This dashboard provides team performance statistics including:

- Median response time metrics
- Messages sent counts
- Tickets resolved numbers
- Performance data for both team members and AI Agents

### Messages Sent by User

Users can click on specific team members or AI Agents to view sent messages, providing insights into AI response patterns and message volume comparisons with human team members. Users can click messages to view full conversation context within the Inbox.
